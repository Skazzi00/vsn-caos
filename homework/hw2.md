## Problem 0: asm-arm/basics/expression

Реализуйте функцию с меткой f, которая вычисляет значение выражения y = Ax² + Bx + C.

Значения A, B, C и x хранятся, соответственно, в регистрах r0, r1, r2 и r3.

Результат вычисления выражения сохраните в регистре r0.

Все аргументы и значение выражения представимы 32-разрядными знаковыми целыми числами.

Использовать оперативную память запрещено — можно использовать только регистры.

## Problem 1: asm-arm/basics/memaccess

Реализуйте на языке ассемблера ARM функцию с сигнатурой:

```c
int summ(int x0, int N, int *X)
```

Функция должна вычислять значение x0 + ∑xᵢ, где 0 ≤ i < N.

## Problem 2: asm-arm/basics/eqsolution

Реализуйте функцию solve с прототипом:

```c
int solve(int A, int B, int C, int D);
```

которая находит минимальное значение x ∈ [0, 254], являющееся одним из корней уравнения: Ax³ + Bx² + Cx + D = 0.

Все аргументы и значение выражения представимы 32-разрядными знаковыми целыми числами.

Не забывайте про сохраняемые регистры; можно использовать для этого память на стеке.

Гарантируется, что на тестовых данных существует целочисленное решение.

## Problem 3: asm-arm/basics/globalvars

Реализуйте функцию `calculate`, которая вычисляет значение выражения: R = (A×B) + (C×D), где A, B, C и D — глобальные переменные типа `uint32_t`, объявленные во внешнем модуле компиляции, а R — глобальная переменная типа `uint32_t` в текущем модуле компиляции.
